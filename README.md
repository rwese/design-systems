# Design Systems

CDN-ready HTML design system examples using semantic HTML and BEM-style CSS classes.

## Files

- `design-system-cdn.css` — reusable stylesheet intended for CDN hosting
- `design-system-cdn-demo.html` — minimal example page using the CDN stylesheet

## Naming convention

Components use a `ds-` namespace with BEM conventions:

```text
.ds-component
.ds-component__element
.ds-component--modifier
```
